User Experience Design

The processes that go into creating exceptional user experiences. This includes understanding and defining the content structure and behaviors of the end users, and is critical to allowing users to achieve their goals, complete tasks, and find what they need to be successful easily. This can include information architecture and interaction design.

Usability Review and Testing

We get the best ideas and inspiration for solving user experience challenges from interviewing and observing end users completing the everyday tasks they perform while interacting with the systems they use. We take our research seriously as it is key to uncovering the best ways to create a great user experience. We employ several methods for user research and select the method(s) for the project depending on project, goals, and timeline. This can include: Heuristic and Expert Reviews, Usability Consulting, Formal Usability Testing, End User Interviews, UX Requirements Analysis, Process Analysis.

User Experience Strategy

As part of our discovery process we analyze your end users, the competitive landscape, your business goals, and short and long term goals for your software product. This helps us to provide the best user experience by focusing on building the right product in the right way. The UX strategy component can include business and user needs assessment, product requirements, competitive analysis, a product evolution plan, and a user experience strategy briefing.

Visual Design

Visual Design communicates your brand on the information structures, workflows, interactivity, and relationships between the elements and components on a screen. It is essential aspect of UX design for applications. The visual design component can include brand exploration, visual design mockups, high fidelity detailed design, end user/stakeholder feedback, design style guide, and detailed design assets.

User Interface Development

UI Development is the implementation of user experience designs through either a prototype or full development. The result is the front-end interface of the application that end users will use and interact with. This deployment component can include building live pages from design comps, front-end markup (HTML, XAML, etc), front-end code (JavaScript, C#, etc), templates and master pages, and cascading style sheets.
